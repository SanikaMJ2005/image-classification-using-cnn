About This Project

Project Overview

This CNN Image Classification project demonstrates the power of deep learning for computer vision tasks. It uses a Convolutional Neural Network trained on the MNIST dataset to recognize handwritten digits with high accuracy.

The project showcases the complete pipeline from data loading and preprocessing to model training, evaluation, and deployment through a user-friendly web interface.

Dataset

MNIST (Modified National Institute of Standards and Technology)

  • 70,000 images of handwritten digits (0-9)
  • 28×28 pixel grayscale images
  • 60,000 training samples
  • 10,000 test samples
  • Single color channel (grayscale)

Model Architecture

The CNN model uses the following architecture:

  • Input Layer: 28×28×1 (grayscale images)
  • Conv2D Layers: Multiple convolution layers with ReLU activation
  • MaxPooling Layers: For dimensionality reduction
  • Dropout Layers: For regularization and preventing overfitting
  • Dense Layers: Fully connected layers for classification
  • Output Layer: 10 units with softmax activation (one per digit)

Technologies Used

  • Backend: Flask (Python web framework)
  • Deep Learning: TensorFlow & Keras
  • Frontend: HTML5, CSS3, JavaScript
  • Image Processing: PIL (Python Imaging Library)
  • Data Science: NumPy, Matplotlib, Scikit-learn

Model Performance

The trained CNN model achieves excellent performance metrics:

  • Accuracy: >95% on test dataset
  • Precision: High precision across all digit classes
  • Recall: Excellent recall for digit recognition
  • Inference Time: <100ms per image

Learning Outcomes

This project covers key concepts in deep learning and computer vision:

  • Convolutional Neural Networks (CNN) architecture
  • Image preprocessing and normalization
  • Model training and evaluation
  • Handling class imbalance and overfitting
  • Web deployment with Flask
  • REST API design for machine learning models

Getting Started

  1. Train the model using python cnn_image_classifier.py
  2. Install Flask: pip install flask
  3. Run the application: python app.py
  4. Open your browser and navigate to http://localhost:5000
  5. Upload an image of a handwritten digit to test the classifier
